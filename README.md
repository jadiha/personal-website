## ✦ about

my personal website — built with pixel art skies, frosted glass terminals, and too much pink. features a scroll-driven journey through a gallery of memories and an interactive terminal where you can learn about me.

---

## ✦ stack

| | |
|---|---|
| **framework** | Next.js 15 (App Router) |
| **styling** | Tailwind CSS + custom CSS |
| **fonts** | Press Start 2P · Geist Mono |
| **animations** | canvas pixel art · scroll lerp · RAF loop |
| **deploy** | Vercel |

---

## ✦ features

- `˚｡` pixel art sky canvas that parallax-scrolls as you move through the page
- `˚｡` scroll-driven transitions: welcome → photo gallery → terminal
- `˚｡` interactive terminal with commands: `about`, `experience`, `projects`, `socials`
- `˚｡` frosted glass UI with pastel pink & peach gradients
- `˚｡` photo gallery with hover titles

---

## ✦ run locally

```bash
npm install
npm run dev
```

open [http://localhost:3000](http://localhost:3000) ✿

---

## ✦ terminal commands

once you scroll to the terminal, try typing:

```
about        → who i am
experience   → where i've worked
projects     → things i've built
socials      → find me online
help         → list all commands
```
